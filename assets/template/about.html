About/Credits/Support

This is a simple map strategy designer for Counter Strike: Global Offensive. It takes heavy influence from a recent (04/05/2015) post on "/r/GlobalOffensive" by FoxlinkThunder. It uses Canvas and AngularJS.

The Code for the project can be found on Github and is licensed under MIT

DISCLAIMER: This site MAY track parts of your usage in order to gather information as to the performance on your client and/or viewing devices. None of this information is related or can be traced back to a person. My intentions with this data are purely scientific in nature as it only monitors changes to the site itself not the behaviour of the person viewing it.
